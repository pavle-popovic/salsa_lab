# The Salsa Lab

Full stack application generated from requirements and design specifications. The Salsa Lab is a gamified online salsa academy featuring structured courses, XP progression, and video feedback.

## Quick Start

To start the entire application (Frontend + Backend + Database) using Docker:

```bash
docker-compose up --build
```

- **Frontend**: http://localhost:3000
- **Backend API Docs**: http://localhost:8000/docs
